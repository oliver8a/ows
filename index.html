Objetivo de OWS (Ontology Web Search):

Tener una aplicación que permita descargar de un sitio WEB ontologías de un dominio específico. Ayuda en gran medida a los investigadores a tener un medio para obtener ontologías acerca de un tema, o conocer mas acerca de un dominio. Muestra todas las partes de la ontología en una interfaz amigable y fácil de usar y entender.






La aplicación te da la opción de poder descargar la ontología en diferentes formatos existentes(JASON, TURTLE, HTML),además de poder descargar la ontología de dbPedia, sitio donde se encuentra toda la información de la Wikipedia, o de w3c sitio relacionado a la web semántica.

QUE ES UNA ONTOLOGÍA?

Una ontología es una base de datos que describe conceptos del mundo o de algún dominio, algunas de sus propiedades y como los conceptos se relacionan unos con otros. Las ontologías permiten trabajar con conceptos, en lugar de palabras clave, en los sistemas de recuperación de información por ejemplo en las búsquedas en internet


COMO SE REPRESENTA?

Se representan con clases, utilizadas para representar entidades o conceptos, instancias, que son la particularización, realización especifica u ocurrencia de una determinada clase, propiedades de objeto, y propiedades de datos

USOS DE LAS ONTOLOGÍAS

Son utilizadas para representar el conocimiento, mejorar la comunicación humano-maquina, permitir compartir el conocimiento, servir de base para lenguajes de representación de conocimiento, servir de uso para la adquisición de la información, etc.
